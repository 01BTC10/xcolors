Contribute

If you want to contribute your own color themes or any themes you collected over time please do so using the project's github repository.

  1. Fork the xcolors repository.
  2. Add your theme file(s) in the themes folder found in the project's root directory
  3. Commit the newly added files.
  4. Make a pull request.
  5. Changes will show here as soon as I push the chages at heroku.

For more info for supported file formats see the README file found in the git repository.
